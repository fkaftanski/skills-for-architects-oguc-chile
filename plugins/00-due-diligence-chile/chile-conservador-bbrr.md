# Skill: chile-conservador-bbrr

## Description
Consulta el Conservador de Bienes Raíces de la comuna indicada y devuelve inscripción, gravámenes, servidumbres, prohibiciones y titular actual.

## Input
- Dirección completa o Rol SII

## Output
- Tabla con: Inscripción Nº, Fecha, Titular actual, Gravámenes, Servidumbres, Prohibiciones.

## Domain knowledge
Usa fuentes oficiales del Conservador correspondiente. Siempre citar fecha de consulta y número de inscripción.
