# Skill: chile-sii-propiedad

## Description
Consulta avalúo fiscal, rol de propiedad, contribuciones, destino y propietario según SII.

## Input
- Rol SII (ej: 123-45-6789) o dirección + comuna

## Output
- Tabla: Rol, Avalúo fiscal 2026, Contribuciones anuales, Destino (habitacional/comercial), Propietario registrado.

## Domain knowledge
Datos actualizados al año fiscal vigente. Incluir advertencia sobre fecha de vigencia del avalúo.
