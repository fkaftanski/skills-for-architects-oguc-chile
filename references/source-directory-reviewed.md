# Source Directory Reviewed

## Core maintained

- **MINVU, OGUC, LGUC, BCN, Diario Oficial** - núcleo normativo. Son la base de respuestas vinculantes o cuasi vinculantes.
- **DDU** - núcleo interpretativo oficial. Mantener cuando se necesite criterio de aplicación.
- **SEC / RIC / Decreto 08** - núcleo técnico-legal para instalaciones eléctricas de consumo.
- **INN** - referencia técnica crítica para normas técnicas.
- **Reglamento Ley 21.442** - núcleo normativo para copropiedad.
- **DS 40/2012 SEIA** - núcleo legal-ambiental para gatillos y evaluación ambiental.
- **Ley 17.288 / Consejo de Monumentos** - núcleo patrimonial.

## Support maintained

- **DOM en Línea** - apoyo operativo para trámites y estado de gestiones, no fuente de la norma.
- **Proveedores Técnicos MINVU** - apoyo operativo para procesos oficiales.
- **IDE / visores IPT / mapas oficiales** - apoyo referencial para localizar instrumentos, no norma por sí mismos.
- **Buscadores oficiales** - apoyo operativo para localizar textos.

## Secondary degraded

- **CChC** - útil para contexto sectorial, no para fijar la regla jurídica.
- **Colegio de Arquitectos** - útil para ética, práctica y contexto profesional, no para norma primaria.
- **AOA** - útil como contexto sectorial.
- **Instituto de la Construcción** - útil como referencia técnica complementaria cuando no exista mejor fuente oficial.

## Excluded from core

- Portales gremiales o de opinión que repiten normativa sin texto oficial.
- Blogs, notas periodísticas y resúmenes no trazables.
- Material promocional o comercial.
- Cualquier fuente que no permita verificar origen, fecha y alcance.

## Notes

- For predial or communal questions, always contrast the official norm with the local instrument before answering categorically.
- For structural, electrical, sanitary, or environmental questions, verify against the competent technical or legal source before concluding.
