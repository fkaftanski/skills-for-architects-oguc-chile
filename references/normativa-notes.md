# Normative Notes

This repository is a support tool for Chilean architectural workflows.

## Source handling

- Treat OGUC, PRC, NCh, DS, and municipal documents as the primary sources.
- Do not hardcode vigency claims unless they are attached to a dated source or the user explicitly requests an example.
- If a municipal rule is not supplied, label it as pending verification.

## Language policy

- Use sober legal language.
- Avoid claims that imply professional authority or final approval.
- Distinguish between regulatory reading, technical inference, and verified municipal response.
