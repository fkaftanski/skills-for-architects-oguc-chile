# Disclaimer Profesional Chile

Este contenido tiene finalidad de apoyo técnico y documental.

- No sustituye la revisión, firma ni responsabilidad de un profesional competente.
- No equivale a permiso, aprobación ni pronunciamiento oficial.
- Si el análisis depende de una fuente no verificada, debe marcarse expresamente como pendiente de verificación.
- Cuando la materia sea urbanística, estructural, eléctrica, sanitaria, ambiental o patrimonial, la conclusión debe contrastarse con la fuente primaria y con el profesional competente antes de usarla como base de decisión.
- Si existe contradicción entre una fuente secundaria y una fuente primaria, prevalece la fuente primaria.
