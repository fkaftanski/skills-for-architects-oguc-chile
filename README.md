# skills-for-architects-oguc-chile

Repositorio base para agentes, plugins, reglas y hooks orientados a practica arquitectonica en Chile, con foco en OGUC, PRC, due diligence, sostenibilidad y normativa tecnica local.

## Estructura

- `.claude-plugin/`: metadata del proyecto/plugin.
- `agents/`: definiciones de agentes especializados.
- `plugins/`: skills y paquetes tematicos.
- `rules/`: reglas operativas y de citacion.
- `hooks/`: scripts de automatizacion y validacion.
- `OGUC-QUICKSTART.md`: guia rapida de uso.

## Estado

Este repo fue inicializado a partir de la estructura objetivo entregada por el usuario. Los archivos Markdown criticos ya quedaron creados en version inicial para poder iterarlos y reemplazarlos facilmente con contenido definitivo.
