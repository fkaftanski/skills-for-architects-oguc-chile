# PRC, Rasantes y Altura

Use this skill for basic massing checks based on PRC and OGUC context.

## Operative steps

1. Identify the site and relevant zoning context.
2. Separate PRC rules from OGUC baseline assumptions.
3. Check:
   - rasantes
   - altura máxima
   - setbacks or retrait
   - envelope conflicts
4. Flag missing municipal source material.

## Output shape

- Executive summary
- Zoning inputs
- Massing constraints
- Conflicts or gaps
- Verification items

## Guardrails

- If the PRC is not provided, do not invent the limit.
- Use a textual diagram when useful, but keep it simple.
