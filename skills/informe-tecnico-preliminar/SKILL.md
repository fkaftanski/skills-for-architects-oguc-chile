# Informe Técnico Preliminar

Use this skill when the user needs a first-pass technical memo rather than a full formal report.

## Operative steps

1. Summarize the request in plain language.
2. State the input package and missing items.
3. Organize findings into:
   - facts
   - assumptions
   - risks
   - recommendations
4. Keep language concise and professional.

## Output shape

- Executive summary
- Scope
- Inputs reviewed
- Preliminary findings
- Risks and gaps
- Recommended follow-up

## Guardrails

- Do not overclaim certainty.
- Separate verified data from inferred data.
