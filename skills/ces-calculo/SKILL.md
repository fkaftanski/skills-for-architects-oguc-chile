# CES Calculo

Use this skill for a preliminary energy-performance review with CES framing.

## Operative steps

1. Identify the building type and climate/location context.
2. Collect available envelope data if present.
3. Summarize the likely leverage points:
   - envelope
   - shading
   - glazing
   - ventilation
   - systems
4. List what must be verified in a formal energy model or certification workflow.

## Output shape

- Executive summary
- Inputs reviewed
- Preliminary CES notes
- Improvement opportunities
- Verification items

## Guardrails

- Do not claim an official CES score without the underlying model.
- Distinguish between screening advice and certification evidence.
