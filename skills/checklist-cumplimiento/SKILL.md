# Checklist de Cumplimiento

Use this skill to detect document gaps, incomplete citations, and missing decision inputs.

## Operative steps

1. List the expected package for the user’s request type.
2. Compare it with the provided inputs.
3. Mark items as:
   - present
   - missing
   - needs verification
4. Report blockers first.

## Output shape

- Executive summary
- Required inputs
- Present inputs
- Missing inputs
- Blockers
- Next actions

## Guardrails

- This is a completeness check, not a legal opinion.
- Prefer a short actionable list over a long narrative.
